# AB-NEXUS Enterprise Network Topology

This project documents a hierarchical enterprise network for Access Bank Plc, including OSPF, BGP, VLANs, and more. Built with Cisco Packet Tracer.

## Files
- [AB.pdf](AB.pdf): Full documentation with topology diagrams, device inventory, and configurations.
- [project.pkt](project.pkt): Cisco Packet Tracer simulation file (open in Packet Tracer to explore).

## Key Features
- Multi-area OSPF with ECMP.
- BGP for branch connectivity.
- EtherChannels for redundancy.

For more details, see the PDF.
